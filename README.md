# Skirack-Feedback

Velkommen! Dette er stedet for å rapportere bugs og 
komme med forslag til SkiRack-appen.

## Hvordan rapportere

1. Klikk på **Issues** øverst på siden
2. Klikk **New issue**
3. Velg riktig template (Bug report eller Feature request)
4. Fyll ut skjemaet så detaljert du kan

## Hva er SkiRack?

SkiRack er en app for å holde oversikt over ski, 
slipinger og testresultater for langrennsski.
